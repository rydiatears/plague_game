# The Plague Year

React/Vite app and standalone artifact for a Reigns-style Black Death classroom game.

## Current scope

- Single-page local app
- Four core stats: Death, Wealth, Order, Faith
- Full historical deck with phase-based sequencing
- Conditional cards via prerequisite choice chains
- Inline historian notes during play plus end-state summary
- Keyboard choice support with `A` and `D`

## Run

With Node.js installed:

```bash
npm install
npm run dev
```

## Single-file version

The primary standalone file is [plague-year-artifact.html](C:/Plague%20Game/plague-year-artifact.html). It opens directly in the browser with no dev server.
